# Emotion Detection Application

This project is a web-based emotion detector built with Python, Flask, and Watson NLP.

## Features
- Detects anger, disgust, fear, joy, and sadness
- Returns the dominant emotion
- Includes error handling for empty input
- Includes unit testing
- Packaged as a Python module

## Project Structure
- EmotionDetection/
- server.py
- test_emotion_detection.py
- templates/index.html
